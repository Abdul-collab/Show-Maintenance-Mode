Put your WordPress site in a customizable maintenance mode with a countdown timer, background image, and message.

== Description ==

**Show Maintenance Mode** allows site administrators to enable a full-page maintenance mode screen with:
- A customizable message
- A countdown timer
- A background image
- Admin panel to manage settings
- Professional UI and settings saved in the database
- Persistent countdown that doesn't reset on refresh

Use it when updating your website or preparing for a relaunch. Lightweight and reliable.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/show-maintenance-mode/` or install it from the Plugin Directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to **Settings > Maintenance Mode** to configure.
4. Click "Start Maintenance" to enable it.

== Frequently Asked Questions ==

= Who can see the maintenance screen? =  
All visitors will see the maintenance page when it's active, except admin users who are logged in.

= Can I customize the background and message? =  
Yes, you can add a background image URL and set a custom message from the settings page.

= Will the countdown persist on refresh? =  
Yes, the countdown timer is stored in the database with an exact end time.
